# PowerUtil — Downloads

**Power tools for your desktop** — a PowerToys-style utility suite for macOS
(launcher, clipboard history, window layouts, color tools, screen capture,
and ~40 more), with one-click in-app updates.

This repo hosts the **prebuilt app + auto-update feed only**. The source lives
in a separate repository.

## Download & install (macOS, Apple Silicon)

1. Go to the [**latest release**](../../releases/latest).
2. Download `PowerUtil_<version>_aarch64.dmg`.
3. Open the DMG and drag **PowerUtil** into **Applications**.
4. First launch: **right-click PowerUtil → Open → Open** (the build isn't
   Apple-notarized, so Gatekeeper warns once).
5. Grant **Accessibility** when asked — needed for global hotkeys, window
   tiling, and mouse/keyboard features. Some tools also ask for **Screen
   Recording** or **Automation** the first time you use them.

## Updates

PowerUtil checks this repo for new versions. When one is available,
**Settings → Updates → Download & install** updates the app in place — no need
to re-download manually.

## Requirements

- macOS on Apple Silicon (arm64).
